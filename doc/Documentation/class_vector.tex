\subsection{Detailed Description}
\doxyref{Vector}{p.}{class_vector} class This class os responsible for computing if a collision happens between two balls. It is class that defines vectors and allows us the use basic vector operations such as addition, subtraction, scalar multiplication, scalar product between two vectors and calculating unit vectors in given direction.
